# Project-Management
Progetto di Project Management e supporto alle decisioni del corso di Laurea Magistrale di Ingegneria Informatica presso l'Università Politecnica delle Marche. 
Il progetto mira alla realizzazione di un centro sportivo all'avanguardia, concepito per ospitare una vasta gamma di discipline, facendo dello sport un punto di incontro fondamentale per la comunità e favorendo lo sviluppo sociale e formativo.
